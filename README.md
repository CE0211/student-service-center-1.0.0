# 校园服务网站定制演示

这是公开展示版本，用于演示手机优先的高校新生导航、阶段任务、互动清单、第一周时间表、安全提醒、常见问答与入口导航能力。

当前版本包含：

- 暖杏、珊瑚、向阳、青芽、晴空、靛蓝、柔紫七套可切换主题
- 主题选择与清单进度浏览器本地保存
- 手机、平板和电脑响应式布局
- 独立的微信、QQ 联系页面
- 无后台、无账号登录、不上传个人清单数据

- 本仓库只包含可公开访问的展示版。
- 原始母版、接单资料和生产流程不在本仓库中。
- 正式项目会复制为独立版本，并部署到客户自己的仓库。
- 本站不对应任何真实学校，不提供真实报到、缴费或登录服务。

线上展示地址：<https://ce0211.github.io/campus-launchpad-demo/>

仓库的 `main` 分支禁止强制推送和删除。正式客户项目不会直接在本展示仓库上修改，而是从本地母版创建新的独立仓库。

首个公开版本已经备份为 Git 标签 `v1.0-original`。
