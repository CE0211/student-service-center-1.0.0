# Student Service Center-1.0.0

这是校园服务站定制网站 `1.0.0` 正式版，用于呈现手机优先的高校新生导航、阶段任务、互动清单、第一周时间表、安全提醒、常见问答与入口导航能力。

当前版本包含：

- 暖杏、珊瑚、向阳、青芽、晴空、靛蓝、柔紫七套可切换主题
- 主题选择与清单进度浏览器本地保存
- 手机、平板和电脑响应式布局
- 独立的微信、QQ 联系页面
- 无后台、无账号登录、不上传个人清单数据

- 本仓库只包含可公开访问的正式站点文件。
- 原始母版、接单资料和生产流程不在本仓库中。
- 正式项目会复制为独立版本，并部署到客户自己的仓库。
- 本站不对应任何真实学校，不提供真实报到、缴费或登录服务。
